\doxysubsection{Detailed Description}
Tube diameter for CLF (phi\+\_\+eq) 

When a certain tube will be accessible for CLF is stored in {\ttfamily t\+\_\+eq\+\_\+ar} as that information is aquired. Always, this time is in the future. This routine interpolates the stored values to return the tube diameter relevant for CLF at the current time.
