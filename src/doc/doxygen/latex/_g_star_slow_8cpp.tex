\doxysubsection{Detailed Description}
Tube relaxation part of relaxation. 

Given some frequency w, returns G\textquotesingle{}(w), G\char`\"{}(w), epsilon\textquotesingle{}(w), epsilon\char`\"{}(w)

Assume G(t)/\+G\+\_\+0 = mu(t) R(t), with mu and R determined by exponential weighted integral of changes in phi and phi\+\_\+\+ST

R(t) is slowly relaxing. Extend R(t) to infinity analytically.

Dielectric relaxation is considered to be proportional to mu(t)
